# Caption Numbering 中文文档
[English version](README.md)
## 功能特性

- **基于标签的自动编号**：自定义标签（如 `图`、`表`、`算法`）。以 `###### 图 xxx` 开头的标题自动编号为 `###### 图 1 xxx`、`###### 图 2 xxx` 等。
- **自定义标签管理**：在设置面板中增、删、改标签。每个标签拥有独立的计数器。
- **章节前缀编号（可选择）**：可选择地从章节标题提取编号作为前缀（如 `## 1 系统概述` 下的题注变为 `###### 图 1-1 xxx`）。
- **智能标签建议**：输入 `###### ` 时自动弹出下拉菜单选择标签。
- **自动编号**：可选的定时自动编号（每 10 秒），保持题注始终最新。
- **双链引用同步**：题注重新编号时，所有指向它的 `[[#caption]]` 双链引用自动更新：
  - `[[#图 1 系统架构]]` → `[[#图 2 系统架构|图 2]]`（目标 + 显示文字同步更新）
  - `[[#图 1 系统架构]]` → `[[#图 1 系统架构|图 1]]`（缺少显示文字时自动补全）
- **Front matter 支持**：通过 YAML front matter 为每个文档独立设置（如 `caption numbering: 图, 表, auto`）。
- **移除编号**：一键命令清除所有题注编号。

## 安装方法

### 从 Obsidian 社区插件安装（推荐）

1. 进入 **设置 → 第三方插件 → 社区插件 → 浏览**。
2. 搜索 "Caption Numbering"。
3. 点击 **安装**，然后 **启用**。

### 手动安装

1. 从 [Releases](https://github.com/zhangyitong625-zjuchem/Caption-Numbering-Obsidian/releases) 页面下载最新版本。
2. 解压到你的库的 `.obsidian/plugins/caption-numbering/` 文件夹。
3. 在 **设置 → 第三方插件** 中启用插件。

## 使用方法

### 基本流程

1. 打开 **命令面板**（`Cmd/Ctrl + P`）。
2. 输入以下命令之一：

| 命令 | 说明 |
|------|------|
| **编号文档中所有图/表题注** | 自动为所有匹配标签的 `######` 题注编号 |
| **移除文档中所有题注编号** | 清除所有题注的数字编号 |
| **将当前标签设置保存到文档 front matter** | 将当前标签设置写入文档 front matter |

3. 你也可以在 **设置 → 快捷键** 中为这些命令分配快捷键。

### 编写题注

有两种方式创建题注：

**方法 A — 使用标签建议弹窗：**
1. 输入 `###### `（六个井号 + 空格）。
2. 自动弹出下拉菜单，显示已配置的标签。
3. 选择一个标签（如 `图`）。
4. 光标自动定位到标签名后，可以继续输入题注内容。
5. 结果：`###### 图 系统架构图`

**方法 B — 手动输入：**
```markdown
###### 图 系统架构图
###### 表 数据对比
###### 算法 快速排序
```

### 执行编号

编写题注后，运行 **编号文档中所有图/表题注** 命令。结果：

```markdown
###### 图 1 系统架构图
###### 图 2 流程图
###### 表 1 数据对比
###### 算法 1 快速排序
```

### 章节前缀编号

如果文档中有已编号的章节标题，如：

```markdown
## 1 系统概述
  ###### 图 系统架构图
  ###### 图 流程图
## 2 功能设计
  ###### 图 部署图
```

在设置中启用 **章节前缀** 模式。结果：

```markdown
## 1 系统概述
  ###### 图 1-1 系统架构图
  ###### 图 1-2 流程图
## 2 功能设计
  ###### 图 2-1 部署图
```

### 使用双链引用交叉引用

你可以使用 Obsidian 内置的双链语法引用题注：

```markdown
[[#图 1 系统架构]]
```

执行编号后，这会自动变为：

```markdown
[[#图 1 系统架构|图 1]]
```

如果题注被重新编号，引用也会自动更新：

```markdown
[[#图 2 系统架构|图 2]]
```

### 每个文档单独设置（front matter）

在文档的 front matter 中添加：

```yaml
---
caption numbering: 图, 表, auto
---
```

这会将标签设为 `图` 和 `表`，并为此文档启用自动编号。

可用选项：
- `图, 表, 算法` — 逗号分隔的标签名
- `auto` — 启用自动编号
- `off` — 关闭此文档的编号

## 设置说明

| 设置项 | 说明 |
|--------|------|
| **标签管理** | 增、删、改题注标签。每个标签有独立的计数器。 |
| **编号模式** | `单级编号` — 图 1, 图 2... / `章节前缀` — 图 1-1, 图 1-2... |
| **标题来源级别** | 从第几级标题提取章节编号（1–5） |
| **标题编号级数** | 取标题编号的前几级（如 `1` → `1`，`2` → `1.2`） |
| **编号分隔符** | 章节前缀与序号之间的分隔符（如 `-`、`.`、`:`） |
| **自动编号** | 开关定时自动编号（每 10 秒） |
